#URL_Shortener

URLShortener (Go)  High-performance URL shortener built in Go. Handles thousands of concurrent requests using goroutines. Features: custom aliases, click analytics, REST API. Tech Go 1.21+ Redis  

Сокращатель URL на Go. Обрабатывает тысячи конкурентных запросов через горутины. Возможности: кастомные алиасы, статистика, REST API. Стек: Go 1.21+, Redis
